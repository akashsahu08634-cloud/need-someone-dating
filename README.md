Need Some One?

Need Some One? is a dating discovery demo website built using HTML, CSS, JavaScript, GitHub, and GitHub Pages.

This project was created for learning and portfolio practice. It is not a real dating platform. All profiles, names, photos, locations, and details used in this project are fictional/demo-based.

Live Website

https://akashsahu08634-cloud.github.io/need-someone-dating/

Project Features

- Girls and Boys profile sections
- Like and Dislike buttons
- 3-star reward after liking a profile
- Search option by name or location
- Profile photos using generated avatar-style images
- Rose petal animation for the girls section
- Premium black and gold theme for the boys section
- Mobile responsive design
- Safety and privacy section
- Hosted live using GitHub Pages

Technologies Used

- HTML
- CSS
- JavaScript
- GitHub
- GitHub Pages

Purpose of This Project

The purpose of this project is to practice front-end web development and learn how to create, edit, publish, and manage a live website using GitHub.

Through this project, I learned how to create a repository, add an HTML file, write CSS styling, use JavaScript for interaction, commit changes, and publish the website online.

Important Note

This is only a demo project. It does not collect user data, does not have real users, and does not use real personal profiles.
